# 🎵 MyTube Playlist Manager

Um gerenciador de vídeos do YouTube moderno e funcional, desenvolvido com foco em organização de conteúdo e facilidade de uso. Ideal para agrupar vídeos por categorias, como estudos, lazer ou rotinas de treino.

## 🚀 Funcionalidades

* **Criação de Quadros (Playlists):** Organize seus vídeos em diferentes categorias com nomes personalizados.
* **Conversão Inteligente de Links:** Suporte para diversos formatos de URL do YouTube (desktop, mobile e encurtados).
* **Drag & Drop (Sortable.js):** Reorganize a ordem dos seus vídeos simplesmente arrastando-os para a posição desejada.
* **Persistência de Dados:** Todos os seus quadros e vídeos são salvos automaticamente no navegador (LocalStorage), garantindo que nada se perca ao fechar a página.
* **Design Responsivo:** Interface otimizada para visualização em computadores, tablets e smartphones.

## 🛠️ Tecnologias Utilizadas

* **HTML5 & CSS3:** Estrutura semântica e estilização moderna com Grid e Flexbox.
* **JavaScript (Vanilla):** Lógica de manipulação do DOM e armazenamento local.
* **Sortable.js:** Biblioteca externa para a funcionalidade de arrastar e soltar.
* **Font Awesome:** Ícones para uma interface mais intuitiva.

## 📦 Como Instalar e Usar

1.  **Clone o repositório:**

2.  **Abra o arquivo `index.html`** diretamente em qualquer navegador moderno.
3.  **Hospedagem no GitHub Pages:**

    * Selecione a branch `main` e a pasta `/ (root)`.
    * Clique em **Save** e acesse a URL fornecida pelo GitHub.
